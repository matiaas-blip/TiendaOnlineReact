# 🎀 Pinkie Nova Gaming

Pinkie Nova Gaming es una tienda online de periféricos y accesorios gamer desarrollada con React, Vite y Tailwind CSS.

El proyecto simula una experiencia de compra moderna inspirada en tiendas de e-commerce, incorporando búsqueda, filtros, carrito de compras y un diseño responsive adaptado para computadoras, tablets y dispositivos móviles.

---

## ✨ Características

### 🛍️ Catálogo de productos

* Visualización de productos en formato tarjetas.
* Diseño inspirado en plataformas de e-commerce.
* Imágenes, precios, categorías y stock.

### 🔎 Búsqueda y filtros

* Búsqueda por nombre.
* Filtrado por categoría.
* Ordenamiento por precio:

  * Menor a mayor.
  * Mayor a menor.

### 🛒 Carrito de compras

* Agregar productos.
* Eliminar productos.
* Aumentar o disminuir cantidades.
* Cálculo automático del total.
* Contador de productos en el Navbar.
* Persistencia mediante LocalStorage.

### 💳 Finalización de compra

* Formulario de datos del comprador.
* Selección de método de entrega.
* Confirmación de compra.
* Banner de compra exitosa.

### 📱 Responsive Design

* Adaptado para:

  * Computadoras.
  * Tablets.
  * Teléfonos móviles.

### 🎀 Personalización visual

* Estética cute y gamer.
* Paleta de colores rosa pastel.
* Diseño moderno y amigable.

## 🚀 Instalación

1. Clonar el repositorio

```bash
git clone URL_DEL_REPOSITORIO
```

2. Entrar a la carpeta del proyecto

```bash
cd tienda-online
```

3. Instalar dependencias

```bash
npm install
```

4. Ejecutar el servidor de desarrollo

```bash
npm run dev
```

5. Abrir en el navegador

```bash
http://localhost:5173
```
